# Pills MVP

Telegram-бот для отметок о приёме нескольких препаратов по календарю. Каждый препарат может иметь своё ежедневное расписание — например, один или два приёма в день.

## Возможности

- добавление нескольких препаратов;
- курс на 7 дней, 1 месяц, 3 месяца, произвольное количество дней или без срока окончания;
- начало курса сегодня или в любую выбранную дату, включая прошедшую;
- приём каждый день или только в выбранные дни недели;
- необязательная заметка о способе приёма препарата;
- от одного до трёх приёмов в день без учёта времени;
- отдельная отметка факта для каждого приёма;
- просмотр и изменение отметок за любую дату через календарь;
- уведомления для схемы 1 раз в день: в 11:00;
- уведомления для схемы 2 раза в день: в 11:00 и 23:00;
- уведомления для схемы 3 раза в день: в 11:00, 17:00 и 23:00;
- удаление препарата;
- раздельные данные для каждого пользователя Telegram.

## Запуск

1. Создайте бота через [@BotFather](https://t.me/BotFather) и получите токен.
2. Подготовьте окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

3. Вставьте токен в `.env`:

```dotenv
BOT_TOKEN=ваш_токен
```

4. Запустите:

```bash
python bot.py
```

Данные сохраняются локально в `pills.db`. Приёмы создаются ежедневно только внутри выбранного срока курса. Бот группирует препараты в одно уведомление на временной слот. Для схемы 1 раз в день уведомление приходит в 11:00, для 2 раз — в 11:00 и 23:00, для 3 раз — в 11:00, 17:00 и 23:00 по часовому поясу из `TIMEZONE`.

Если пользователь не нажал ни одну кнопку для приёма, он переносится в следующую запланированную группу уведомлений этого дня. Явный ответ «Ещё нет» прекращает перенос, не отмечая приём.
